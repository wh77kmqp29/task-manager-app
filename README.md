# タスクマネージャーアプリ

## 概要
日々のタスクを管理できるWebアプリケーションです。

## なぜ作ったか
（ここに自分の体験を書く。例：）
日常のやるべきことを忘れてしまうことが多く、
シンプルで直感的に使えるタスク管理ツールが欲しいと思い作成しました。

## 機能一覧（予定）
- タスクの追加・編集・削除
- 優先度の設定（高・中・低）
- カテゴリ分け
- 期限の設定

## 使用技術（予定）
| 種類 | 技術 |
|------|------|
| フロントエンド | HTML / CSS / JavaScript |
| バックエンド | Python |
| データベース | SQLite |
