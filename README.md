# Daily Report Manager

## 概要
業務日報と健康チェックを記録・管理できるWebアプリケーションです。
ブラウザだけで動作し、データはローカルに保存されます。

## なぜ作ったか
日々の体調や業務内容を手軽に記録・振り返りできるツールが欲しいと思い作成しました。

## 機能一覧
- 健康チェック（体温・体調・気分の記録）
- 業務日報（日ごとの作業内容を記録）
- 月別の表示切り替え
- 統計バー（記入済み日数・好調日数の表示）
- PDF出力（印刷・保存用）
- バックアップ・リセット機能
- レスポンシブ対応（スマホでも使える）

## 使用技術
| 種類 | 技術 |
|------|------|
| フロントエンド | HTML / CSS / JavaScript |
| データ保存 | localStorage |
| PDF出力 | ブラウザ印刷機能 |

## 技術的なポイント
- 単一HTMLファイルで完結（サーバー不要）
- localStorage でデータを永続化
- タブ切り替えUIを自前で実装
- レスポンシブデザイン対応

## 使い方
1. `index.html` をブラウザで開く
2. ユーザー名と対象月を設定
3. 健康チェックまたは業務日報を記入
4. PDF出力ボタンで印刷・保存
